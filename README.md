MediVoice 

MediVoice is an Android-based medical assistance application that helps users with symptom tracking basic health guidance and emergency support features.

  Features
- Symptom tracking system
- Basic health guidance
- Emergency contact support
- User-friendly interface


 Technologies Used
- Java
- XML
- Android Studio
- Gradle


 Project Structure
- `app/` → Main application code
- `res/` → UI resources (layouts images etc.)
- `AndroidManifest.xml` → App configuration
